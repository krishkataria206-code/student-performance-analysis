Projects Repository

This repository contains the work completed as part of my data analytics and machine learning projects. The projects focus on analyzing datasets, applying machine learning techniques, and presenting insights through code, reports, and presentations.

Project 1: Student Performance Analysis

In this project, we analyzed a dataset related to student performance. The objective was to understand how different factors such as study habits, attendance, and other variables affect student academic performance.

The dataset was cleaned and analyzed using Python libraries such as Pandas, NumPy, and Matplotlib. Exploratory Data Analysis was performed to identify patterns and relationships in the data. The results were summarized and presented through visualizations and a report.

Files related to this project include the Jupyter Notebook with the analysis code and the summary report explaining the findings.

Project 2: Telecom Customer Churn Prediction

This project focuses on predicting customer churn in the telecommunications industry using machine learning techniques. The goal was to analyze telecom customer data and build a model that can predict whether a customer is likely to discontinue the service.

The project involved data preprocessing, exploratory data analysis, customer segmentation using K-Means clustering, and building predictive models such as Logistic Regression and Random Forest. The results were evaluated using performance metrics and visualizations.

The repository includes the dataset, Jupyter Notebook, project report, and presentation related to this analysis.
